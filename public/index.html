Featured Applications


Chatty, The Chat Bot

Project Description

A chat bot that you can speak to. Web Speech API transcribes spoken statements by the user, Socket.io sends transcriptions between the client and the server and Dialogflow AI (API.AI) interprets the statements that the server receives from the client and formulate responses. Web Speech API then takes the returned text from Dialogflow and speaks those words. The icon font sets of Themify, Icofonts and Weather Icons are used to enhance the responses from the AI. A Google Cloud Platform function is used to handle webhooks from Dialogflow in order to get the weather.

Technology

  • Themify, Icofont & Weather Icon Icon Fonts
  • Web Speech API - Recognition and Synthesis
  • World Weather Online API
  • Socket.io
  • Dialogflow AI
  • Google Cloud Platform
  • Node.js and Express.js
